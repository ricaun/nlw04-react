## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Create Project

```bash
yarn create next-app projectName
```

## Update to Typescript

```bash
yarn add typescript @types/react @types/react-dom @types/node -D
```

## Cookies

### Install
```bash
yarn add js-cookie
```

### Install TypeScript
```bash
yarn add @types/js-cookie -D
```
